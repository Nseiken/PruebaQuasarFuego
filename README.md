# PruebaQUasar

## Instalación local

Usar la última versión de node posible [Node.js](https://nodejs.org/).
Clonar el proyecto
```sh
git clone https://github.com/Nseiken/PruebaQuasarFuego.git
```
Instalar dependencias
```sh
cd PruebaQuasarFuego/
npm i
```

Servimos la aplicación

```sh
node app
```
